# スタイルフォルダ

このフォルダには、アプリケーションのスタイルシート（CSS）が格納されています。

## ファイル構成

### App.css
メインアプリケーションのスタイルシートです。

## スタイルの構成

### 基本スタイル
- コンテナ、タイトル、説明文
- ボタン（プライマリ、セカンダリ、ログ）
- セクション、フィールド
- 入力フィールド

### コンポーネントスタイル
- アコーディオン
- エラーメッセージ、警告メッセージ
- バリデーション表示
- CalcLineカード
- サマリー表示
- 保存一覧

### レスポンシブデザイン
- モバイル対応（`max-width: 640px`）
- タブレット対応
- デスクトップ対応

## カラーパレット

詳細は `design/UI_DESIGN.md` の「カラーパレット」セクションを参照してください。

**主要カラー**:
- プライマリ: `#2563eb` (Blue 600)
- セカンダリ: `#64748b` (Slate 500)
- エラー: `#ef4444` (Red 500)
- 警告: `#f59e0b` (Amber 500)
- 成功: `#10b981` (Green 500)

## スタイルの適用

スタイルは以下の方法で適用されます：

1. **CSSクラス**: インラインスタイルをCSSクラスに置き換え
2. **コンポーネント**: Reactコンポーネントに`className`属性を設定
3. **インポート**: `App.tsx`で`import './styles/App.css'`として読み込み

## スタイルのカスタマイズ

スタイルを変更する場合：

1. `App.css`を編集
2. 必要に応じて新しいCSSクラスを追加
3. コンポーネントの`className`属性を更新
