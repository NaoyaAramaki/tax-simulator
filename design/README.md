# 設計書フォルダ

このフォルダには、納税金額シミュレーターの設計書が格納されています。

## ファイル構成

### REQUIREMENTS.md（要件定義書）
システムの要件を定義したドキュメントです。
- プロジェクト概要
- 機能要件（FR-001～FR-060）
- 非機能要件（NFR-001～NFR-013）
- 制約事項（CON-001～CON-008）
- システム境界
- 用語定義
- 注意事項・免責事項

### BASIC_DESIGN.md（基本設計書）
システム全体のアーキテクチャと基本設計を記載したドキュメントです。
- システムアーキテクチャ
- 技術スタック
- モジュール構成
- データ構造設計
- モジュール設計
- UI設計
- データフロー
- エラーハンドリング
- セキュリティ・パフォーマンス考慮事項

### DETAILED_DESIGN.md（詳細設計書）
各モジュールの詳細な実装仕様を記載したドキュメントです。
- 計算エンジン詳細設計
- バリデーション詳細設計
- データ保存詳細設計
- ルール管理詳細設計
- UI詳細設計
- 計算式詳細
- エラーハンドリング詳細
- テスト仕様

### UI_DESIGN.md（UIデザイン仕様書）
ユーザーインターフェースのデザイン仕様を記載したドキュメントです。
- デザインコンセプト
- カラーパレット
- タイポグラフィ
- レイアウト
- コンポーネントデザイン
- インタラクション
- レスポンシブデザイン
- アクセシビリティ
- ユーザビリティ
- 実装例

### SPECIFICATION.md（統合仕様書）
上記の設計書を統合した包括的な仕様書です（参考用）。

## ドキュメントの関係性

```
REQUIREMENTS.md（要件定義）
    ↓
BASIC_DESIGN.md（基本設計）
    ↓
DETAILED_DESIGN.md（詳細設計）
    ↓
UI_DESIGN.md（UIデザイン）
```
