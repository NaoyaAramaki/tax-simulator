# 納税金額シミュレーター

所得税・住民税・ふるさと納税の納税額を計算するWebアプリケーションです。

## 機能

- 給与所得、事業所得、株式所得の計算
- 各種控除（基礎控除、配偶者控除、扶養控除、社会保険料控除など）の計算
- 所得税・住民税の計算
- ふるさと納税の最適化計算
- 計算過程の詳細表示
- 結果の保存・読み込み機能

## 技術スタック

- **フロントエンド**: React + TypeScript
- **ビルドツール**: Vite
- **テスト**: Vitest
- **デプロイ**: Vercel

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# テスト実行
npm run test
```

## デプロイ

このプロジェクトはVercelにデプロイできます。

1. GitHubリポジトリにプッシュ
2. Vercelでプロジェクトをインポート
3. ビルドコマンド: `npm run build`
4. 出力ディレクトリ: `dist`
